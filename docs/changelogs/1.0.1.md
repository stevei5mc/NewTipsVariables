# **1.0.1**
---
## **注意事项**
1. **从该版本起（1.0.1）配置文件的自动更新不会受到配置文件的控制，如果配置文件有新的内容会自动执行更新**
2. **不要随意修改配置文件的版本号，否则带来的后果自负（虽然从1.0.1版本起会对被重置的配置文件生成备份文件，但还是不要随意修改）**
3. **建议备份好已经配置好的配置文件以备不时之需**
