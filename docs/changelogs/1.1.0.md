# **1.1.0**
---
## **注意事项**
1. **从版本 `1.0.1` 配置文件的自动更新不会受到配置文件的控制，配置文件如果有新的内容将会自动执行更新**
2. **从版本 `1.0.1` 起会对配置文件的版本号有异常的配置文件进行重置，在重置时会对旧配置文件进行备份**
3. **不要随意修改配置文件的版本号，否则配置文件的更新机制所带来的后果需要自负**
4. **建议备份好已经配置好的配置文件，以便在配置文件自动更新时出现问题可以及时恢复**
