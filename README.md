# NewTipsVeriable
## 插件介绍
> 该插件可以为Tips添加更多的变量内容，更多变量敬请期待
## 使用方法
> 1、将插件放进`plugins`文件夹  
> 2、需确保安装`Tips`插件后重启服务器  
> 3、在`/plugins/NewTipsVariables`文件夹中的txt文件获取相关变量介绍，也可在本页面获取相关的变量介绍
